ContextMesh Documentation

Persistent semantic memory for AI agents. Write context from any agent, query it from any other — across tools, sessions, and your whole team.

◈
Base URL: https://api.contextmesh.dev · All requests require Authorization: Bearer cm_live_...

Quickstart

Get running in under 2 minutes. Get your API key from contextmesh.dev, then:

# pip install contextmesh
from contextmesh import Mesh

mesh = Mesh("cm_live_your_key_here")

# Store context from any agent
id1 = mesh.remember("Prod DB is postgres 15.3, AWS us-east-1", tags=["database"])
id2 = mesh.remember("Use camelCase for JS, snake_case for Python", tags=["conventions"])

# Query it from any other agent — semantic, not keyword
hits = mesh.query("what should I know before touching the database?")
for h in hits:
    print(h["score"], h["text"])
# → 0.94  Prod DB is postgres 15.3, AWS us-east-1
// npm install contextmesh
import { Mesh } from "contextmesh";

const mesh = new Mesh("cm_live_your_key_here");

await mesh.remember("Prod DB is postgres 15.3, AWS us-east-1", { tags: ["database"] });

const hits = await mesh.query("what should I know about the database?");
console.log(hits[0].text, hits[0].score);
# Store context
curl -X POST https://api.contextmesh.dev/remember \
  -H "Authorization: Bearer cm_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{"text":"Prod DB is postgres 15.3","tags":["database"]}'

# Query it
curl -X POST https://api.contextmesh.dev/query \
  -H "Authorization: Bearer cm_live_your_key" \
  -H "Content-Type: application/json" \
  -d '{"q":"database info?","top_k":5}'
// ~/.cursor/mcp.json  OR  ~/.claude/mcp.json
{
  "contextmesh": {
    "command": "npx",
    "args":   ["contextmesh-mcp"],
    "env":    { "CM_KEY": "cm_live_your_key" }
  }
}

// Claude/Cursor now has tools:
// remember(text, tags?) — store context
// recall(query, top_k?) — semantic search
// forget(id)            — delete entry
// list_context()        — browse all

Authentication

All requests must include your API key in the Authorization header:

Authorization: Bearer cm_live_your_key_here

API keys start with cm_live_ for production and cm_test_ for test environments. Get your key at contextmesh.dev.

⚠
Never commit your API key. Use environment variables: export CM_KEY=cm_live_...

POST /remember

Store a piece of context. The text is embedded into a 1536-dimension vector and stored with your metadata.

POST /remember Store context

Request body

ParameterTypeRequiredDescription
textstringrequiredThe context to store. Max 10,000 characters.
tagsstring[]optionalTags for filtering. e.g. ["database", "infra"]
source_agentstringoptionalWhich agent wrote this. e.g. "cursor", "claude"
confidencefloatoptionalCertainty score 0.0–1.0. Default 1.0

Response

{
  "id": "e1a2b3c4-..." // entry ID — use to delete later
  "status": "stored"
  "chars": 42 // character count of stored text
  "tags": ["database"]
}

POST /query

Semantic search over your workspace context. Returns the top-K most relevant entries ranked by cosine similarity score.

POST /query Semantic search

Request body

ParameterTypeRequiredDescription
qstringrequiredNatural language query. Max 2,000 characters.
top_kintoptionalMax results. Default 5, max 20.
tag_filterstringoptionalOnly return entries with this tag.
min_scorefloatoptionalMinimum similarity threshold 0.0–1.0. Default 0.3

Response

{
  "query": "what do we know about the database?"
  "count": 2
  "results": [
    {
      "id": "e1a2b3c4-..."
      "text": "Prod DB is postgres 15.3..."
      "score": 0.9312 // cosine similarity 0–1
      "tags": ["database", "infra"]
      "source_agent": "cursor"
      "created_at": 1700000000 // unix timestamp
    }
  ]
}

DELETE /forget/:id

Remove a specific context entry by its ID. Get the ID from /remember or /list.

DELETE /forget/:id Delete entry

Returns {"id":"...","status":"deleted"} on success. Returns 404 if not found.

GET /list

Browse all stored entries in your workspace. Paginate with limit and offset. Filter by tag with tag_filter.

GET /list Browse entries
Query paramTypeDefaultDescription
limitint50Max entries (1–200).
offsetint0Skip N entries for pagination.
tag_filterstring—Only return entries with this tag.

GET /usage

Returns this month's usage stats for your workspace and plan limits.

GET /health

Liveness check. Returns {"status":"ok","service":"contextmesh","ts":...}. No auth required.

Python SDK

pip install contextmesh

Sync client — Mesh

from contextmesh import Mesh, ContextMeshError
from contextmesh._errors import AuthError, RateLimitError

mesh = Mesh("cm_live_...")

# remember(text, tags=[], source_agent=None, confidence=1.0) → str
entry_id = mesh.remember("prod DB postgres 15", tags=["database"])

# query(q, top_k=5, tag=None, min_score=0.3) → list[dict]
hits = mesh.query("database?", top_k=3)

# forget(entry_id) → bool
mesh.forget(entry_id)

# list(limit=50, offset=0, tag=None) → dict
result = mesh.list(tag="database")

# usage() → dict
u = mesh.usage()

# remember_many(items) → list[str]
ids = mesh.remember_many(["entry one", "entry two"])

Async client — AsyncMesh

from contextmesh import AsyncMesh

async def main():
    mesh = AsyncMesh("cm_live_...")
    await mesh.remember("API rate limit is 1000 rpm")
    hits = await mesh.query("rate limits?")
    ids  = await mesh.remember_many(["a", "b", "c"])  # concurrent

MCP Server

The MCP server exposes ContextMesh as a Model Context Protocol server, so any compatible AI tool can use it natively without writing any integration code.

ℹ
Python required. The MCP server runs as a Python process. You need Python 3.10+ installed.

Tools exposed

ToolDescription
rememberStore context. Args: text (required), tags, confidence
recallSemantic search. Args: query (required), top_k, tag
forgetDelete by ID. Args: id (required)
list_contextBrowse stored entries. Args: limit, tag

Guide: Cursor Integration

// 1. Install
npm install -g contextmesh-mcp   // or use npx (no install)

// 2. Add to ~/.cursor/mcp.json
{
  "contextmesh": {
    "command": "npx",
    "args":   ["contextmesh-mcp"],
    "env":    { "CM_KEY": "cm_live_..." }
  }
}

// 3. Restart Cursor. The agent now has remember/recall tools.

Error Handling

StatusErrorCause
401AuthErrorInvalid or revoked API key
404NotFoundErrorEntry ID not found in this workspace
422ValidationErrorMissing required field or invalid value
429RateLimitErrorRate per minute or monthly quota exceeded
500APIErrorInternal server error — contact support
 
from contextmesh._errors import AuthError, RateLimitError, ContextMeshError

try:
    mesh.remember("test")
except AuthError:
    print("Check your API key")
except RateLimitError:
    print("Quota hit — upgrade or retry in 60s")
except ContextMeshError as e:
    print(e.status_code, str(e))

Self-Hosting

△
OpenAI key required for self-hosting. The embedder uses text-embedding-ada-002. Without a key it falls back to a deterministic hash-based embedder that is NOT semantically meaningful — queries won't return relevant results.
